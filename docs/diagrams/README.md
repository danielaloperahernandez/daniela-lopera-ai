# Diagrams

The architecture diagrams are written inline as Mermaid in each README (they render on
GitHub), which keeps them versioned with the code.

If you want exported images (for slides or the website), drop PNG/SVG exports here, e.g.:

- `overall-architecture.png` - the system overview from the root README
- `doc-intelligence-flow.png` - Project 1 hybrid flow
- `support-agent-flow.png` - Project 2 agent flow
- `rag-pipeline.png` - Project 3 retrieval/generation

To export: paste the Mermaid block into <https://mermaid.live> and download as PNG/SVG.
